Markdown的使用小结
==================
### 一、标题
    # 一级标题
    ## 二级标题
    ### 三级标题
    #### 四级标题
    ##### 五级标题
    ###### 六级标题
    大标题
    ==   (大于两个等于号就可以，有的一个等于号夜可以，但是以防万一)
    小标题
    --  （同上）

**预览效果：**
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
大标题
==
小标题
--

### 二、粗体、斜体

```
***粗体***
___粗体___
*斜体*
_斜体_

```

**预览效果：**  

**粗体**   
__粗体__   
*斜体*
_斜体_  
>（像上面不自动换行的时候，可以“空两格”达到换行的效果）

### 三、分割线

```
---
***
~~文字删除线~~

```
**预览效果：**
---
***
~~文字删除线~~

###六、添加表格
```
|header 1 | header 2 |
|---|---:|
|row 1 col 1 | row 1|
|row 2 col 1 | row 2 col 2|


header 1 | header 2 
---|---
row 1 col 1 | row 1
row 2 col 1 | row 2 col 2

:----:   为剧中
:----    为靠左
----:    为靠右

```

**预览效果：**  

|header 1 | header 2 |
|---|---:|
|row 1 col 1 | row 1|
|row 2 col 1 | row 2 col 2|


header 1 | header 2 
---|---
row 1 col 1 | row 1
row 2 col 1 | row 2 col 2

###七、添加代码块
```
｀一段文字｀

｀｀｀
代码段
｀｀｀
```
**预览效果：**
```python
a = 100
if a >= 0:
    print(a)
else:
    print(-a)
```

###八、引用
```
>引用的文字  
>引用的文字  
>引用的文字  

---  (如果下面还要继续引用，最好要分隔开，要不然会连在一起)

>引用的文字引用的文字  
引用的文字  
引用的文字

---

>引用的文字 

 >>引用的文字 
 （引用内二次引用，上下最好都回车下，要不然
 下面会被连起来当成二次引用的内容）
 
>引用的文字

```
**预览效果：**
>引用的文字  
>引用的文字  
>引用的文字  

---  

>引用的文字引用的文字  
引用的文字  
引用的文字

---

>引用的文字 

 >>引用的文字 
 
>引用的文字  

###九、首行缩进
```
&nbsp;缩进一个字符  (&nbsp;半角的不断行空格)
 
&ensp;&ensp;缩进两个字符  (&ensp;半角的空格)
  
&emsp;&emsp;缩进四个字符 (&emsp;全角的空格)

```
**预览效果：**  
&nbsp;缩进一个字符  
 
&ensp;&ensp;缩进两个字符  
  
&emsp;&emsp;缩进四个字符

###十、添加脚注

```
A[^1]  
[^1]:fdfdfdfd
```
A[^1]  
[^1]:fdfdfdf

###十一、创建链接
为输入的url或邮箱自动创建链接，如:test@domain.com  
```
<test@domain.com>
```
<test@domain.com>

###十二、转义字符
在特殊字符，如*、[、>等前面加\可使特殊格式字符转换为正常字符打出（有序列表符号如1.，须在. 前加\）。

```
\\
\`
\*
\_
\{\}
\[\]
\(\)
\#
\+
\-
\.
\!
```
